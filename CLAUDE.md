# G1 프로젝트 Claude 규칙

## 코드 주석 규칙

- 모든 public 함수/메서드에는 목적, 파라미터, 반환값을 설명하는 주석 작성
- 복잡한 로직, 조건문, 알고리즘에는 인라인 주석 포함
- Unity 이벤트 메서드(Awake, Start, Update 등)에는 해당 메서드의 역할 주석 작성
- 새로 작성하는 코드뿐만 아니라 수정하는 코드에도 주석 유지
- 모든 주석은 한국어로 작성 (인라인 주석, XML 문서 주석 포함)

## 코드 작성 규칙

- 언어: 주석은 반드시 한국어 사용, 변수명/함수명은 영어 사용, 사용자와의 대화는 한국어로
- 네임스페이스: 새 스크립트 작성 시 프로젝트명(G1) 또는 기능별 네임스페이스 사용
- 파일 인코딩: 모든 스크립트는 UTF-8로 저장
- Unity 버전: 6000.4.0f1 기준으로 작성 (최신 API 사용 권장)
- 플랫폼: Android 타겟 기준

## Unity 작업 규칙

- 씬 수정 후에는 저장 여부 확인
- 새 컴포넌트 추가 후 컴파일 오류 확인 (read_console 활용)
- 서드파티 에셋 스크립트 수정 시 반드시 네임스페이스 추가
